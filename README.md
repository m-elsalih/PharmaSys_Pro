نظام إدارة الصيدلية الذكي - نظام متكامل لإدارة الأدوية والمخازن.

📝 نبذة عن المشروع
مشروع PharmaSys_pro هو تطبيق مكتوب بلغة Python مصمم لمساعدة الصيادلة في إدارة عملياتهم اليومية بكفاءة عالية. يتضمن التطبيق واجهة مستخدم رسومية (GUI) سهلة التعامل، مع قاعدة بيانات مدمجة لحفظ كافة البيانات بشكل آمن.

✨ المميزات الرئيسية
إدارة المخزون: إضافة، تعديل، وحذف بيانات الأدوية.
نظام الفواتير: إصدار فواتير بيع وحفظها بصيغة رقمية.
قاعدة بيانات مدمجة: يستخدم التطبيق pharma_system.db لتخزين البيانات.
واجهة مستخدم احترافية: تم تصميم الواجهة باستخدام مكتبات Python الحديثة لتوفير تجربة مستخدم سلسة.
📁 هيكل المجلدات
يحتوي المشروع على المجلدات والملفات التالية:

models/: تحتوي على هياكل البيانات.
ui/: ملفات واجهات المستخدم.
database/: ملفات قاعدة البيانات.
invoices/: المجلد المخصص لحفظ الفواتير الصادرة.
main.py: الملف الرئيسي لتشغيل البرنامج.
